# 异星工厂产线计算工具（Factorio 2.0 原版）

一个针对 **异星工厂 Factorio 2.0（无模组）** 的产线计算工具，支持 **整条生产链递归计算**，并提供 **Tkinter 图形界面**。

你只需要选择目标物品、目标产量和组装机等级，程序就会自动算出：

- 每种中间产物需要多少台机器（组装机 / 炉子 / 化工厂 / 炼油厂 / 离心机等）
- 一直到矿石、煤、水、原油、石油气等为止的 **原料输入需求（每秒）**

---

## 功能特性

- **整链递归计算**：从目标物品（例如：红电路、绿瓶、蓝瓶、火箭部件等）一路拆到基础原料。
- **机器数量计算**：按配方时间和设备速度，计算每种中间产物需要多少台机器，自动向上取整。
- **组装机等级可选**：在界面中选择使用组装机 1 / 2 / 3，会影响所有 `assembling` 类配方的速度。
- **支持多种物品类别**：包括电路板、科研包、油品与化工品、模块、物流、建筑、武器、防御、火箭链路等，大部分原版配方都已覆盖。
- **简单易用的 GUI**：使用 Tkinter 实现，无需手算或记配方，适合一边开游戏一边查产线数据。

---

## 环境要求

- 操作系统：Windows / macOS / Linux 均可（本工程在 Windows 10 上开发）
- Python：建议 Python 3.10 及以上
- 依赖：只使用 Python 标准库（`tkinter`、`math` 等），无需额外安装第三方包  
  > 注意：有些精简版 Python 发行版可能没有自带 `tkinter`，这种情况需要单独安装。

## 安装与运行

### 1. 获取项目代码

如果你是从 GitHub 上获取项目，可以在命令行执行：

```bash
git clone https://github.com/mario02222227/factorio-production-calculator.git
cd factorio-production-calculator
```

### 2. 运行程序

在项目根目录下执行：

```bash
python peiping.py
```

如果你有多个 Python 版本，可能需要用：

```bash
python3 peiping.py
```

运行后会弹出一个图形界面窗口。

## 使用说明

1. **选择目标物品**
   - 在界面中选择目标物品，例如：
     - 红电路（高级电路）
     - 绿瓶（物流科研包）
     - 蓝瓶、紫瓶、黄瓶
     - 各种中间件（齿轮、电路板、塑料、酸等）
2. **设置目标产量**
   - 输入一个数字（例如 `1`、`2.5`、`60`）
   - 选择单位是「每秒」还是「每分钟」
3. **选择组装机等级**
   - 选择使用组装机 1 / 2 / 3  
   - 程序会根据对应的 `crafting speed` 自动计算
4. **点击计算**
   - 程序会显示：
     - 每种中间产物对应的机器数量（按类型区分）
     - 各类原料（矿石、煤、水、石油气等）的每秒需求量
   - 结果可以用来规划总线、炼油区、矿区规模等。

## 配方与逻辑说明

- **配方数据来源**  
  `RECIPES` 字典基于 Factorio 2.0 原版数据整理，尽量与游戏内一致。
- **机器速度**  
  - 组装机：`ASSEMBLING_SPEED = {1: 0.5, 2: 0.75, 3: 1.25}`
  - 炉子、化工厂、炼油厂、离心机等使用近似速度常量，方便统一计算。
- **计算核心**  
  - 单配方机器数量由 `calculate_assemblers(...)` 完成；
  - 整条链路由 `compute_requirements(...)` 递归完成，从目标物品一路向下拆分。

## 修改与扩展

- **修改已有配方**
  - 在 `RECIPES` 中找到对应 key（例如 `logistic_science_pack`）
  - 调整 `time`（配方时间）、`output`（每次产出数量）、`inputs`（原料字典）
- **新增配方 / 物品**
  - 按现有写法增加一条 `Recipe(...)`
  - 如果需要在 GUI 中可选，还需要把该物品 key 和显示名称加入到物品列表 / 分类配置中。
